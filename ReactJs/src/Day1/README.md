# Trainning ReactJs

## JSX & Component

### JSX

- Javascript syntax extension: cú pháp mở rộng từ js
- Trình duyệt kh thể hiểu file.jsx - cần công cụ để chuyển đổi sang js --> Vite
- Cần môi trường nodejs chạy vite

### Component

- Là khối chứa HTML/CSS và JS.
  --> Tái sử dụng: Dùng nhiều lần, kh lặp lại code
  --> Dễ dàng quản lý: Chia nhỏ giao diện phức tạp thành các phần nhỏ
  --> Tập trung mã liên quan: HTML/CSS & JS liên quan trong một component
- Cách khai báo: Là một hàm của JS tuân theo các quy tắc
  - Tên function: Viết hoa chữ cái đầu
  - Phải return về một giá trị hiển thị và là 1 phần từ duy nhất
  - VD: function App(){
    return (
    <>TODO</>
    )
    }
